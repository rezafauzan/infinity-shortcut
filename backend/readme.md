# Infinity Shortcut RESTAPI Backend
App to short your long link into short one

## Architecrure
using dependency injection layered application architecture serving RESTful API

## Dependencies
- github.com/gin-gonic/gin
- github.com/jackc/pgx/v5
- github.com/joho/godotenv
- github.com/golang-jwt/jwt/v5
- github.com/matthewhartstonge/argon2

## How To Run
- Simply download the source code 
- go run ./cmd/main.go
- or wait release version
- open in browser http://localhost:8888
